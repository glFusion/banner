glFusion Banner Plugin Category Configuration

Category ID
Every category required an ID. You may enter your own or one will be automatically created.
Category
An identifier for this category. This short name will be appear in lists to identify the category.
Type
Enter a string to identify the type for this category. This string will be used to determine the ad placement in templates, blocks and content.

For template variables, use the name of the template itself. For example, to have a category place your ad in the Featured Story, set the "type" to "featuredarticle". Then, in <layout_dir>/featuredstorytext.thtml, add a new template variable "{banner_featuredarticle}". This variable will be replaced with the banner content.

Description
Enter a description for this category.
Topic
Select the topic with which this category's ads will appear.
Max. Image Width
Enter the maximum allowed width for images in this category. If left blank, the global configuration item will prevail.
Max. Image Height
Enter the maximum allowed height for images in this category. If left blank, the global configuration item will prevail.
Enabled?
If this is unchecked, then no banners will be shown for this category.
Centerblock?
Check this box to have this category's ads show in a centerblock
Mapping Enabled?
Check to send ads from this category to the specified template.
Mapping Position
Enter the position for the ad in index pages, e.g. after the first item, second item, etc. Leave at zero to prevent ads from showing in this list. Be sure to check "show in content" if you want the ad shown in the main item content. If "show once" is not checked this is used as an interval.
Show Mapping Item Once?
Check this to have the ad displayed only once on an index page. If unchecked, the "position" is used as an interval to show ads every 2nd item, every 3rd item, etc.
Mapping Content Area
Check this to show the ad in the main content area, for templates that support this.
Visible To
Select the user group that can view ads in this category. Normally this will be "All Users" but you may wish to limit ad visibility in some cases.
